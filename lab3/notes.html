What is the correct way for HTML tags to nest?

All tags should be closed properly and completely. For example, the body tag is nested within the html tag.


How do you add an id to an HTML tag?

In order to add an id tag to an HTML element, place the word id followed by the equal sign and your id name within the opening set of tags.

HEADER ID


How do you add a class to an HTML tag?

In order to add a class to an HTML element, place the word class followed by the equal sign and your class name within the opening set of tags.

CLASS ID


Why would you use an ID vs a class?

You would use a class when you have several elements which you would like to be styled in the same fashion. The ID attribute is used for unique instances within the HTML document.

How do you calculate the width and height of an element?

The width and height of an element is calculated using pixels, or a percentage of the page. For example, when calculating using pixels it is done like this:


150px * 150px 200px * 150px 250px * 150px

When calculating the width and height of an element using a percentage it looks like this:


30% * 30% 70% * 30%
30% * 70% 70% * 70%

Explain two situations where margins collapse.

Collapsing margins happen when two vertical margins come in contact with one another. If one margin is greater than the other, then that margin overrides the other, leaving one margin.
